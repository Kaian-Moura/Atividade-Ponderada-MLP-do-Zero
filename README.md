# MLP do Zero — Classificação de Dígitos MNIST

Implementação de um Multi-Layer Perceptron (MLP) do zero, usando apenas NumPy para operações matriciais.

## Estrutura do Projeto

```
.
├── README.md              ← Este arquivo
├── requirements.txt       ← Dependências do projeto
├── mlp/
│   ├── __init__.py
│   ├── network.py         ← Implementação do MLP
│   ├── activations.py     ← Funções de ativação e suas derivadas
│   ├── losses.py          ← Cross-entropy e outras funções de loss
│   └── optimizers.py      ← SGD e otimizadores opcionais
├── notebooks/
│   └── experimentos.ipynb ← Experimentos e análises
└── results/
    └── (plots, tabelas, figuras geradas)
```
